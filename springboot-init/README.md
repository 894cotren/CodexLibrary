## 模版使用步骤
1. 全局替换模块名，由springboot-init改为自定义项目名
2. 全局修改包名，重构com.grey.springbootinit+自定义
3. 全局搜一下修改前的包名，免得有漏网之鱼
4. 退出idea 从文件夹里删除.idea然后再进入idea加载一下，免得maven的项目还是springboot-init
5. 去除掉不需要的模块，然后愉快使用；
